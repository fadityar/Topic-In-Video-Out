# Topic-In-Video-Out

Increase content production efficiency by up to 10x.

Topic-In-Video-Out is a static GitHub Pages landing page for an Automated Kids Story Generator. The product concept turns a story idea into a complete MP4 video by coordinating AI story planning, consistent character and background generation, scene assembly, narration, subtitles, transitions, and background music.

## Website

The site is designed to run directly from GitHub Pages:

- `index.html`
- `styles.css`
- `script.js`
- `favicon.svg`

No build step is required.

## Local Preview

Open `index.html` directly in a browser, or serve the folder with any static server.

## GitHub Pages

This repository includes a GitHub Actions workflow at `.github/workflows/pages.yml` that deploys the repository root as a static GitHub Pages site.
